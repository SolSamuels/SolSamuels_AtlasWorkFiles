This directory contains the most updated and versatile version of RBF_NN.py
and it's example Slurm submission script, RBF_NN.sh. This script can be changed
to study various Diatomic Molecules (and diatomics only) with various number of
states in order to determine the ensemble atom-in-molecule decomposition for the 
system as a function of internuclear separation.
